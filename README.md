## Getting Started

The following tasks are supported via [Rake](https://github.com/ruby/rake).

```bash
bundle install
bundle exec rake db:create 
bundle exec rake app:run 
bundle exec rake db:drop 
```

To run tests:

```bash
bundle exec rake spec
```
